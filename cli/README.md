## CLI Client Skeleton (`cli/`)

This directory will contain the **command-line uploader** for batch transcription.

### Planned responsibilities

- Authenticate with the backend.
- Upload local audio files (e.g. `meeting.wav`) for batch transcription.
- Poll job status and fetch transcripts and summaries.

Example (planned):

```bash
transcriber upload meeting.wav
```

Implementation will be added later; this is only the project skeleton.
