**O1 — §4.1 vs текущий `GET /admin/api/v1/conversations`:**

| §4.1 (обзор списка) | Статус v1 |
|---------------------|-----------|
| Фильтр по статусу ASR / активного транскрипта | **Сделано** — `transcript_status`, `transcript_kind` |
| Фильтр по статусу сводки сессии (`recording_session_summaries`) | **Сделано** — `session_summary_status`, `session_summary_missing` |
| Фильтр «есть аудио» / очередь загрузки | **Сделано** — `has_audio` |
| Фильтр по цепочке `recording_session_id` | **Сделано** — query `recording_session_id` |
| Размер/длительность аудио как числа без дорогих запросов | **Не в v1** — в ORM нет колонок длительности/размера без S3 HEAD; нужна схема БД или кэш |
| Строгий список query-параметров | **Сделано** — неизвестные ключи → 422 (`parse_list_conversations_query`) |
